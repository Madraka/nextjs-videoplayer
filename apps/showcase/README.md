# Showcase App (Planned)

This folder is reserved for the Next.js showcase application in the monorepo migration.

Current production showcase still runs from the repository root.
## Environment

Copy `.env.example` to `.env.local` and adjust values as needed.

- `NEXT_PUBLIC_STREAM_URL`: default stream URL for the demo page.
- `NEXT_PUBLIC_POSTER_IMAGE`: optional poster image override.
- `NEXT_PUBLIC_GA_ID`: optional Google Analytics ID.
- `NEXT_PUBLIC_PLAYER_DEBUG_LOGS`: set `true` to enable package debug logs.
- `NEXT_PUBLIC_PLAYER_INFO_LOGS`: set `true` to enable package info logs.
